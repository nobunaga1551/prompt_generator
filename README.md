# prompt_generator
prompt generator

## 誰向け？
 - 複雑なタスクを構造化したい人
 - チームで同じフォーマットを使いたい人
 - 毎回同じ構成で書きたいが手間を省きたい人
 - 「なぜか精度が低い」を改善したい人


---

## 変更内容

**PC（641px以上）:** 従来通りのテーブルレイアウト

**スマホ（640px以下）:** テーブル行を**カード形式**に切り替え

```
┌──────────────────────────┬───────┐
│ <Objective>  [必須]      │ ▲  ▼ │
├──────────────────────────┴───────┤
│ 内容を入力…                      │
│                                  │
│ (textarea フル幅・高さ確保)       │
└──────────────────────────────────┘
```

| 対応箇所 | 内容 |
|---|---|
| テーブル行 | CSS Grid でカード化。`内容`欄がフル幅になる |
| ボタン | ▲▼を横並び・40px角でタップしやすく |
| カスタム入力欄 | 縦積みでボタン全幅 |
| ツールバー | 縦積みで全ボタン全幅 |
| 出力エリア | コピーボタン全幅 |
